Introduction

Thank you for purchasing Website Template. Within this manual we will show you how to manage a template, edit it, set up and create a ready-to-use website. We did our best to simplify this instruction and make it clear and easy to use.
Preparation
Before proceeding with Website Template installation, you need to prepare. It is recommended that you follow the steps listed below:
Software
-First, you'll need proper unzipping software.You may use WinZip 9+ (Windows) and Stuffit Expander 10+ (Mac). This is a mandatory software..
-You should also use code editor, i. e. Sublime Text, Notepad++, Atom, etc. to edit the source code of template files.
-For files uploading to server FTP manager will be also needed, like FileManager at your hosting control panel or desktop applications, such as Total Commander, FileZilla, CuteFTP, etc
Files structure
After template unzipping you'll see the following files & folders structure:
bootstrap - includes all bootstrap files, used in template.
css - includes all .css files, used in template.
font - is the Internet's icon library and toolkit, used by .html files.
image - includes all img, used in .html files.
js - includes JavaScript libraries and jQuery plugins.
aboutus.html, cart.html, category.html, checkout.html, contact.html, index.html, login-register.html, my-account.html, payment-policy.html, privacy-policy.html, product.html, return-policy.html, shipping-policy, terms.html, wishlist.html - file represents separate template html page.
favicon.ico - favicon files.
index.html - index.html page is the most common name used for the default page.
documentation.html - here is all the information with instructions on how to install, activate, edit your template.

How to Uploading template

-In order to make your site "live", you need to upload all the content of "carrental" folder from your local computer to your hosting server.
-Please note: your website root depends on the directory structure on your hosting server. If you upload the "carrental" folder itself into your server, the root to your website will be http://your_domain_name/carrental. To avoid this and make the website root http://your_domain_name/ please open the "carrental" folder and upload the folder content in your website root directory.
Upload the files to your host using free FTP software FileZilla.
**If you don't know how to do this, you can look for video tutorials - how i can upload files to your host using free FTP software FileZilla.

How to Edit template

.html and .css files editing
.html files editing
All of editable HTML files are located at "carrental" folder (index.html, menu.html, contact.html, etc.). Every .html file represents separate template page.
To open .html file with Sublime Text 3 application(or with your text editor), you should right mouse click on the file, and in the context menu select Open with > Sublime Text 3(or with your text editor). When the file is opened you can start editing it.
Sublime Text 3(or with your text editor) is a recommended application as it allows you to edit the HTML files content.
